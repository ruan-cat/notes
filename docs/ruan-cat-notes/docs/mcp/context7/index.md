# context7

据说是一个能够索引最新内容的工具。

- https://github.com/upstash/context7

本地版：

```json
{
	"mcpServers": {
		"context7": {
			"command": "cmd",
			"args": ["/c", "npx", "-y", "@upstash/context7-mcp"]
		}
	}
}
```

在线云端版本：

```json
{
	"mcpServers": {
		"context7": {
			"url": "https://mcp.context7.com/mcp"
		}
	}
}
```
