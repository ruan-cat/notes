# ux ，Python 的包管理器

- 仓库： https://github.com/astral-sh/uv

据说除了是包管理器，还是 Python 的版本管理器。

## 安装

```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
