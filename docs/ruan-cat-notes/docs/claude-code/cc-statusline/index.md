## cc-statusline ，设置 statusline 状态栏

想实现自定义的状态栏。实时看到自己的 api 开销。

- 仓库： https://github.com/chongdashu/cc-statusline

在当前项目内初始化配置文件：

```bash
npx @chongdashu/cc-statusline@latest init
```

预览效果：（暂时无效）

```bash
cc-statusline preview .claude/statusline.sh
```
