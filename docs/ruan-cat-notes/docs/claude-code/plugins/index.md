# claude code 插件商城

claude code 支持安装插件了，这里将常用的 claude code 插件列举出来，便于在全新的电脑或开发环境内快速安装：

## ui-ux-pro-max-skill

- 仓库： https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

```bash
/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill
/plugin install ui-ux-pro-max@ui-ux-pro-max-skill
```

## superpowers

- 仓库： https://github.com/obra/superpowers

```bash
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
```

## ruan-cat monorepo

- 仓库： https://github.com/ruan-cat/monorepo

```bash
/plugin marketplace add ruan-cat/monorepo
/plugin install common-tools@ruan-cat-tools
```
