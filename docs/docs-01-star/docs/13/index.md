# 13

自我介绍，糕鸭果，曾用名`阮喵喵`。这是我第六次参加 01 星球的大项目了。

## 我参与本次项目的预期目标

1. 完成最复杂的模块。重点放在移动端上面。
2. 项目无条件使用 pnpm+monorepo 架构，为三个前端项目做统一的整合。
3. 使用 nitro v3 开发一个全栈的项目。使用 nitro v3 开发接口。
4. 使用 neon 数据库存储数据。
5. 使用 drizzle 作为 ORM。
6. 使用 vercel 和 cloudflare worker 作为项目部署平台。前期使用 vercel、后期逐步增加 cloudflare worker。
7. 使用 vitepress 搭建文档项目。
8. 重点学习文件处理，重点在本次项目学习文件上传和下载的知识。重点攻克这个不熟悉的知识点。
9. 重点学习图表制作。重点学习 echart 的知识点，重点学会 echart 的响应式渲染和重新渲染问题。
10. 重点学习报表制作。学会制作一个复杂的中台统计报告。
11. 重点学习服务器部署前端项目。我有 2 个低性能的美国服务器，需要学习在 nginx 内完成前端项目部署和反向代理的东西。
