2. 学会从现有的、成熟的后台项目框架内，将特定特色的一款功能，提炼出一个可以复用的 claude code 子代理，并用这个独立的，跨项目的子代理，赋能其他项目。
3. 在 AI 方面，专门学习了如何制作一个简单的 claude code 任务通知工具。
